# AboutMe API
API for [AboutMe-Page](https://github.com/HypedDomi/AboutMe-Page)

## Run Locally
Clone the project
```bash
git clone https://github.com/HypedDomi/AboutMe-API
```

Go to the project directory
```bash
cd AboutMe-API
```

Install dependencies
```bash
npm install
```

Update the Password in the server.js file

Start the server
```bash
npm start
```
